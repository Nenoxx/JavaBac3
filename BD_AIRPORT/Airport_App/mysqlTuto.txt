Lancer/Couper/Restart la BDD :
	->sudo service mysql start
	->sudo service mysql stop
	->sudo service mysql restart
Ouvrir une connexion MySQL : 
	->mysql -u NOM_USER -p
Sélectionner une BD :
	-> use NOM_DB; (ici ça sera use BD_AIRPORT;)
Fermer la connexion à la DB :
	mysql>QUIT
Créer une table :
	->create table NOM_TABLE (champ1 type1, champ2 type2, ...);
Ajouter une clé primaire
	->alter table NOM_TABLE add primary key (NOM_PROPRIETE);
Ajouter une clé étrangère
	->alter table NOM_TABLE add constraint NOM_CONTRAINTE foreign key (NOM_COLONNE) references NOM_TABLE(NOM_COLONNE);
Exporter la schéma de la BDD :
	->mysqldump -u [username] -p [database name] > [database name].sql
	le flag -d permet d'exporter sans les données.
Ajouter la DB via Netbeans :
	->voir page 173 de Java II
Créer un user basique
	->create user nom identified by 'password';
	->grant all privileges/update/select/delete/... on NOM_DB.NOMTABLE to nom;
enlever tous les privilèges
	->revoke all privileges, grant option from nom;
